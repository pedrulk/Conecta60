# Dicas do Celular para a Terceira Idade

Projeto final da disciplina CMP1044 Algoritmos - PUC Goiás 2026/1.

## Objetivo

O programa ajuda pessoas da terceira idade com dicas simples sobre uso do celular, redes sociais, internet e segurança digital.

## Funcionalidades

- Ver dicas por categoria
- Buscar dicas por palavra-chave
- Enviar uma dúvida
- Ver dúvidas cadastradas
- Salvar dúvidas em arquivo

## Linguagem usada

C
